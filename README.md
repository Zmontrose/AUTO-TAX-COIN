# 🚀 Auto Tax Token (Solana - Token 2022)

An automatic tax token built on Solana using:

- Anchor Framework
- Token-2022
- Transfer Hook Extension

This token enforces taxation on **every transfer** automatically.

## 🔥 Tokenomics

- 5% total tax per transfer
  - 2% Burn 🔥
  - 3% Treasury 💰
- Fully enforced via Transfer Hook
- Cannot be bypassed
- Built for Devnet testing
